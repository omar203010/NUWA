## نسق – مساعد الحركة الذكي

تطبيق واجهة ويب تجريبية يعرض مساعدة ذكية للحركة والتنقل في المواقع الحكومية والفعاليات. يركز على تقديم تجربة مستخدم بسيطة باللغة العربية، مع بيانات تجريبية تبرز تدفق الشاشات والميزات الأساسية.

### الفكرة
- إرشاد المستخدم لأفضل الأوقات لزيارة المواقع وتقليل الازدحام.
- عرض حالة الازدحام والمواقف وحالة الطرق بشكل بصري وسريع.
- تنبيهات ذكية وتوقعات ذروة اليوم.
- تسجيل الدخول/التسجيل محلياً (محفوظ في المتصفح) لإظهار بيانات المستخدم في الملف الشخصي.

### القيمة للمستخدم
- اختصار الوقت عبر معرفة حالة الازدحام قبل التحرك.
- تقليل المخاطر عبر تنبيهات مبكرة عن الحوادث أو الامتلاء.
- تجربة عربية مبسطة مع خرائط ورسوم بيانية مباشرة.

### المكونات الرئيسية
- **الأونبوردنج**: شرائح تعريفية مختصرة تسبق الدخول للتطبيق.
- **المصادقة**: تسجيل/دخول محلي، وحفظ بيانات المستخدم في `localStorage`.
- **الرئيسية**: بطاقات إحصائية سريعة، قائمة مواقع قريبة، خريطة مصغّرة.
- **الخريطة**: عرض المواقع والعلامات مع تمييز الازدحام والمواقف (Leaflet).
- **التنبيهات**: بطاقات تنبيهات مع فلترة (الكل/عاجل/مقروء).
- **تفاصيل الموقع**: مخطط دونات للازدحام الحالي وخط للتوقعات (Chart.js).
- **الإبلاغ عن حادث**: تدفق مبسط لإرسال حادث.
- **الملف الشخصي**: يعرض اسم/بيانات المستخدم من `localStorage` مع خيار تسجيل الخروج.

### التقنيات المستخدمة
- HTML/CSS/JS (فانيلا).
- Leaflet للخرائط.
- Chart.js للرسوم البيانية.
- تخزين محلي عبر `localStorage` (لا يوجد باك-إند حقيقي).

### التشغيل محلياً
1) افتح الملف `index.html` مباشرة في المتصفح.
2) جرّب تدفق الأونبوردنج ثم شاشة الدخول/التسجيل.
3) بعد الدخول ستصل للرئيسية، ويمكنك التنقل بين الخريطة، التنبيهات، الملف الشخصي.

### ملاحظات
- البيانات الحالية تجريبية (mock) لعرض التصميم والتدفق.
- يمكن استبدال ملف الشعار بـ `logo.png` الموجود في جذر المشروع.
